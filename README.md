# Q3 – Tag Browser + Filters

## Overview
A simple Tag Browser screen that uses FlowRow and FlowColumn to display chips that wrap when they run out of space. Users can tap tags to select them and see their selection update in real time. Used a different Material 3 components and modifiers for this page.

## Material 3 Components Used
Scaffold, TopAppBar, FilterChip, AssistChip, HorizontalDivider, Text

## Modifiers Demonstrated
Uses fillMaxSize, fillMaxWidth, padding, size, verticalScroll, and Arrangement.spacedBy(8.dp) - for consistent spacing. Visual state change through primaryContainer color and a checkmark icon.
